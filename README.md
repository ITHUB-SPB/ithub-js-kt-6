## Тема практической работы: взаимодействие с API 

**Выполните задания согласно блокам TODO в кодовой базе репозитория, а именно**
1. Отфильтруйте товары по условию, если их количество будет больше 10. (1 балл)
2. Найдите значение элемента массива (товара), чья цена находится в диапазоне от 5000 до 6000 включительно. (1 балл)
3. Отсортируйте данные товаров по цене (по убыванию). (1 балл)
4. В столбце с оценками выводить не массив всех оценок, а средний балл и их количество. (1 балл)
5. \* Сохранять сортировку при сбросе фильтрации, сохранять фильтрацию при сбросе сортировки. (1 балл)

**Бонусное задание (дополнительные баллы к другим кт)** 
- сымитировать задержку ответа бэкендом с помощью таймаута длительностью от 500 до 5000 мс (2 балла),
- сымитировать возможность отказов бэкенда с вероятностью 1 к 5 (2 балла),
- во время задержки отображать нотификацию типа pending, в случае ошибки - error (2 балла).
